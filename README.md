# Next-Movie

Basic NextJS version 13 (use App Router)

### Festures

- All Movie Popular
- Movie Detail

### Build With

- NextJS
- TypeScript
- Tailwind CSS
- DaisyUI

### Movie API

[themoviedb](https://www.themoviedb.org/)

### SetUp Cypress

- install cypress

```
npm install --save-dev cypress
```

- add run command to package.json

```json
"cypress:open": "cypress open"
```

- update tsconfig.json

```json
"moduleResolution": "Node",
"baseUrl": ".",
```
